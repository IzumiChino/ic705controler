# 内嵌式软键盘使用说明

## 快速开始

### 1. 打开呼号输入

在卫星跟踪界面，找到右侧的"呼号记录"区域，点击输入框：

```
┌─────────────────────────────────────┐
│ 呼号记录 (3)              [清空]    │
├─────────────────────────────────────┤
│ 点击输入呼号...              [✓]    │ ← 点击这里
└─────────────────────────────────────┘
```

### 2. 使用内嵌键盘输入

点击后，内嵌键盘会从下方展开：


输入2个字符后，搜索建议会显示在输入框下方：

### 4. 选择建议或继续输入

- **点击建议**: 直接填入呼号，键盘自动关闭
- **继续输入**: 使用键盘继续输入完整呼号
- **点击完成**: 保存呼号记录

## 功能说明

### 按键功能

| 按键 | 功能 | 说明 |
|------|------|------|
| A-Z | 输入字母 | 自动转为大写 |
| 0-9 | 输入数字 | 用于呼号中的数字部分 |
| ←退格 | 删除字符 | 删除最后一个字符 |
| 完成 | 保存记录 | 保存呼号并关闭键盘 |

### 输入框功能

| 操作 | 功能 |
|------|------|
| 点击输入框 | 显示/隐藏键盘 |
| 点击 ✓ 按钮 | 保存呼号记录 |
| 从CW历史拖拽 | 自动填入呼号 |

### 搜索建议

- **触发条件**: 输入≥2个字符
- **显示数量**: 最多10个建议（2行×5列）
- **匹配规则**: 
  - 完全匹配优先
  - 开头匹配次之
  - 包含匹配最后
- **选择方式**: 点击建议直接填入

## 使用技巧

### 1. 快速输入

```
输入 "BH" → 显示建议 → 点击 "BH6AAP" → 自动填入
```

### 2. 手动输入

```
B → H → 6 → A → A → P → 点击"完成"
```

### 3. 修改输入

```
输入错误 → 点击"退格"删除 → 重新输入
```

### 4. 从CW历史拖拽

```
CW历史中的呼号 → 长按拖拽 → 拖到输入框 → 自动填入
```

### 5. 关闭键盘

```
方式1: 点击输入框再次点击
方式2: 点击"完成"按钮
方式3: 选择搜索建议
```

## 常见问题

### Q1: 为什么不能使用系统输入法？

A: 为了避免系统输入法弹出时遮挡搜索建议，我们使用了内嵌式键盘。这样搜索建议始终可见，提供更好的用户体验。

### Q2: 如何输入小写字母？

A: 呼号规范要求使用大写字母，因此键盘只支持大写输入。所有输入会自动转为大写。

### Q3: 可以输入特殊字符吗？

A: 呼号只包含字母和数字，因此键盘不提供特殊字符输入。

### Q4: 搜索建议不显示怎么办？

A: 搜索建议需要满足以下条件：
- 输入至少2个字符
- 呼号库中有匹配的呼号
- 确保呼号库文件已正确加载

### Q5: 如何更新呼号库？

A: 在设置中找到"呼号库管理"，可以导入自定义呼号库文件。

### Q6: 键盘可以自定义布局吗？

A: 当前版本使用固定的QWERTY布局，未来版本可能会支持自定义布局。

## 注意事项

1. **输入规范**: 只能输入大写字母A-Z和数字0-9
2. **搜索延迟**: 搜索建议会在输入后立即显示，无延迟
3. **记录保存**: 点击"完成"或"✓"按钮才会保存记录
4. **键盘状态**: 键盘状态不会在界面切换时保持
5. **兼容性**: 支持Android 8.0及以上版本
